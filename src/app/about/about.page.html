About App

This App is in beta version.

Certified Users

App allows certified users to add Events(Assignments/News/Notes) and manage events.
Normal user can view Events/Notes and get notification about the new events added.

Events

Events like Assignments/Notes/News varies with the class so make sure you're logged into your respective classes

Notification Reminder

App will send 2 notification before the submission date of assignment if the assignment is not done by the user.
